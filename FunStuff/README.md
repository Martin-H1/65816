# FunStuff

Random experiments, demos, tests, and small projects written in 65816 assembly.

This directory is a collection of miscellaneous programs that don't really fit anywhere else in the repository. Some files may be simple hardware tests, proof-of-concepts, optimization experiments, weird ideas, or just things written for fun while learning the 65816.

## Contents

Examples of what may appear here:

- Small standalone assembly programs
- CPU instruction tests
- Timing experiments
- Graphics or sound demos
- Memory management experiments
- Utility routines
- Half-finished ideas
- "What happens if I do this?" programs

## Notes

- Code quality and documentation may vary between projects.
- Some programs may target different hardware or emulator setups.
- Expect inconsistent structure — this folder is intentionally a sandbox.

## Requirements

Depending on the specific program, you may need:

- The CA65 65816 assembler
- An emulator or real hardware
- Platform-specific ROM/header files
- Additional build scripts or tools

See comments inside individual source files for details.

## Purpose

This directory exists mainly for experimentation, learning, and preserving interesting snippets of 65816 assembly code that might be useful later.

Sometimes the best way to understand a processor is to write strange little programs for it.
